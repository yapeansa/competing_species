```math
\begin{align}
    &x' = x\,(3 - 2\,x - 2\,y), \\
    &y' = y\,(2 - x - y).
\end{align}
```

### Determinando as x-nullclines

As x-nullclines são os pontos do plano em que $x\\,(3 - 2\\,x - 2\\,y) = 0$, isto é, as x-nullclines são: $x = 0$ e $2\\,y = 3 - 2\\,x$.
